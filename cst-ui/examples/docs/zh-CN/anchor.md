## Anchor 锚点

用于跳转到页面指定位置。

### 基础用法

:::demo 最简单的用法。
```html
<Anchor show-ink>
  <AnchorLink href="#home" id="home" title="导航" />
  <AnchorLink href="#inout" id="inout" title="数据录入" />
  <AnchorLink href="#show" id="show" title="数据展示" />
  <AnchorLink href="#text" id="text" title="反馈" />
</Anchor>
<script>
export default {

}
</script>
```
:::
