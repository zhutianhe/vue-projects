## BackTop 返回顶部

当页面内容冗长，需要快捷返回顶部时使用，一般放置在页面右下角位置。

### 基础用法

:::demo 默认位置距离页面右部和底部 30px，滚动至距顶端 400px 时显示。
```html
<template>
  <div style="height: 2000px;">
    向下滚动时自动出现
  </div>
  <BackTop></BackTop>
</template>
<script>
export default {

}
</script>
```
:::
